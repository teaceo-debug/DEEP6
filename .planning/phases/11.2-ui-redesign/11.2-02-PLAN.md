    - `typeAFlashKeyframes` — `{ scale: [1, 1.08, 1], filter: [...three filter strings for normal → intensified glow → normal] }`, duration 1400ms total (200ms flash + 1200ms decay)
    - `radialBloomKeyframes` — keyframes for a `<circle>` expanding r: 90→200 and opacity: 0.3→0 over 1200ms
    - `reducedMotion(variant)` — if `window.matchMedia("(prefers-reduced-motion: reduce)").matches`, returns a zero-duration variant
    - Export a `SIGNAL_BIT_CATEGORIES: ReadonlyArray<CategoryKey>` of length 44 mapping each SignalFlags bit index (0–43) to its category. Executor: inspect `deep6/types` / `dashboard/types/deep6.ts` to find the canonical bit → category mapping used by the backend. If the mapping is not already expressed in types, hand-write it from the bit layout documented in CLAUDE.md / ROADMAP (absorption=ABS-01..07, exhaustion=EXH-01..08, imbalance=IMB-01..09, delta=DELT-01..11, auction=AUCT-01..05, trap=TRAP-01..05, volume=VOLP-01..06, ml=ENG-07/E10 slot). The total must be 44.

    **dashboard/components/score/ConfluencePulse.tsx** — "use client". Props: none (self-selects from store).

    Reads from tradingStore (use `useTradingStore` with a shallow selector):
    - `score` (total_score: number 0–100)
    - `signalFlags` (44-bit bitmask — a number or bigint; executor reads actual field name from store)
    - `tier` ("TYPE_A" | "TYPE_B" | "TYPE_C" | "QUIET")
    - `direction` (-1 | 0 | 1)
    - `categoryScores` (Record<CategoryKey, number>) — per-category 0–100. If the store doesn't expose this directly, derive it by counting fired bits per category from `signalFlags` and `SIGNAL_BIT_CATEGORIES` and normalizing by category size × 100.

    Render a 320×320 `<svg viewBox="0 0 320 320">` centered in its container.

    **Outer ring (radius 150, stroke width 8):**
    - 44 `<path>` arcs, each 8.18° wide with 0.5° gaps (so 8.18 + 0.5 = ~8.68 per slot; verify 44 × 8.68 ≈ 381.9 — too wide; recompute: total 360°, 44 slots ⇒ 360/44 = 8.18° per slot; take 7.68° arc + 0.5° gap per slot). Use 7.68° arc, 0.5° gap.
    - For each arc index i: compute start/end angle; use SVG arc path. Stroke = `--rule` if bit i is unfired, else the category color from `SIGNAL_BIT_CATEGORIES[i]`.
    - Wrap each arc in a `motion.path` that animates stroke on ignite (200ms ease-out to bright; use `AnimatePresence` or `animate` prop with key tied to bit state). Stagger by `arcStagger(i)`.

    **Middle ring (radius 130, stroke width 8):**
    - 8 `<path>` sectors (45° each). Category order (clockwise starting from 12 o'clock): absorption, exhaustion, imbalance, delta, auction, volume, trap, ml.
    - Fill color = category color; opacity = `categoryScores[cat] / 100`. Use `motion.path` with `animate={{ opacity }}` transition 300ms.

    **Inner core (radius 100):**
    - Center number via `<text>` or foreignObject-wrapped `<div>` — prefer foreignObject `<div>` so CSS classes (`text-display`, `tnum`, `glow-lime`) apply cleanly.
    - Number: `score` rounded to integer. Color: `--lime` if ≥80, `--amber` if 50–79, `--text-mute` if <50. When ≥80, apply `glow-lime` class.
    - Digit-roll: use Motion `useMotionValue` + `animate(motionValue, score, digitRollTransition)` then subscribe via `useTransform(mv, v => Math.round(v).toString())` rendered into the text node. On reduced-motion, skip animation and render directly.
    - Below number: tier badge — bordered `<div>` (no fill), 1px `border` in tier color, padding 4px 8px, `text-md` 600 in tier color: `"TYPE_A"` / `"TYPE_B"` / `"TYPE_C"` / `"QUIET"`.
    - Below badge: direction glyph — solid filled triangle SVG: ▲ in `--ask` for +1, ▼ in `--bid` for -1, `─` in `--text-mute` for 0. 18×18px.

    **TYPE_A flash orchestration:**
    - Track `lastTier` in a ref. On transition into TYPE_A (or when `score` crosses 79→≥80), fire the flash:
      1. Set a local state `flashActive` true for 1400ms
      2. When `flashActive`: render outer arcs with stroke `#ffffff` for the first 200ms (via conditional), then settle to category color
      3. Scale inner core group via `motion.g` with `typeAFlashKeyframes` transform
      4. Render a `motion.circle` at center as radial bloom with `radialBloomKeyframes`
      5. Apply a one-shot screen-shake by dispatching a custom event `deep6:shake` on `document.body` — a tiny CSS keyframes animation attached to `body.shake` (4px translate, 80ms). Add the keyframes and `.shake` class to globals.css via a small appendix if needed. Respect `prefers-reduced-motion` (skip shake).
    - All flashes are gated on `!reducedMotion`.

    Fixed-sized 320×320 container. Must render correctly when `signalFlags = 0` (all arcs `--rule`, number `0`, tier QUIET).

    **Loading/empty state** (UI-SPEC §8): if `score` is undefined/null, render `─` in `--text-mute` for the number and all arcs unlit. No spinner.

    Do NOT modify `tradingStore`. Only read.
