## Fixes Applied

### D-01: ScoreWidget right sidebar collapses
- **File:** `dashboard/components/score/ScoreWidget.tsx` line 74
- **Fix:** Added `shrink-0` to ScoreWidget root div. Flex default `flex-shrink: 1` was allowing the 240px column to compress even though `w-60` sets the initial width.
- **Result:** Right column holds 240px non-negotiably at all viewport widths.

### D-02: SignalFeed overlaps chart area
- **File:** `dashboard/app/page.tsx` line 62
- **Fix:** Added `overflow-hidden` to the 3-column row container. Without it, any child that internally grows wide (canvas resize event, unclipped flex item) could cause the whole row to reflow, pushing ScoreWidget off-screen.
- **Result:** All flex children are clipped to the container; no content escapes the column boundaries.

### D-03: Header strip text smaller than UI-SPEC 13px
- **Files:** `dashboard/app/globals.css`, `dashboard/components/layout/HeaderStrip.tsx`
- **Fix 1:** Added `html { font-size: 13px !important }` rule in globals.css, placed before the combined `html, body {}` rule. Tailwind v4 `@import "tailwindcss"` injects preflight that sets `html { font-size: 100% }` (browser default 16px); the `!important` override wins.
- **Fix 2:** Added `style={{ fontSize: '13px' }}` inline to the HeaderStrip `<header>` element as a cascade-proof belt-and-suspenders override that no Tailwind class can beat.
- **Result:** Header strip text renders at exactly 13px regardless of cascade order.

### D-04: FootprintRenderer DPR audit
- **File:** `dashboard/lib/lw-charts/FootprintRenderer.ts`
- **Finding:** Both `ctx.font` assignment sites already use `Math.max(8, Math.round(12 * vpr))` — correct. All coordinate math multiplies by `vpr`. `ctx.save()` / `ctx.restore()` pairs properly bracket font assignments.
- **Action:** Added DPR invariant comment at top of `draw()` documenting the bitmap-space contract for future maintainers.
- **Result:** No regression; contract is now documented.
