<decisions>
## Implementation Decisions

### Locked Bugs (must fix)
- **D-01:** **ScoreWidget right sidebar collapses** at standard desktop width. The 240px right column is not being reserved — it disappears or gets clipped. Root cause likely in `app/page.tsx` flex layout: the 3-column shell (chart `flex-1` | signals/tape 320px | score 240px) needs `shrink-0` on both fixed columns plus `min-width` on the chart column.
- **D-02:** **SignalFeed overlaps the chart area.** Current layout has signal pills floating at top-right of the chart rather than residing in the middle-column signals panel. Likely `position: absolute` on SignalFeed or the signals panel lost its width. Fix: ensure SignalFeed renders inside the 320px column, not as an overlay.
- **D-03:** **Header strip text renders far smaller than UI-SPEC's 13px.** Root cause: Tailwind v4 `@theme inline` tokens or the `text-[13px]` class not being applied; possibly browser-default font-size cascade issue or the header has an ancestor with a small `font-size`.
- **D-04:** **Footprint cells render with tiny text.** Canvas is in bitmap coordinate space — the text draw calls likely use `scope.horizontalPixelRatio` / `verticalPixelRatio` multipliers for box positions but the `ctx.font` size is still in CSS px, so text appears half-size on Retina. Fix: multiply `ctx.font` pixel size by `scope.verticalPixelRatio` when in bitmap space.

### Verification
- **D-05:** After fixes, reload the dashboard and fire `scripts/demo_broadcast.py`. Screenshot must show: score widget visible right column with CONFLUENCE SCORE number readable, signal feed inside middle column (not overlapping), header-strip text clearly legible, footprint cell numbers readable.

### Claude's Discretion
- Exact CSS mechanism (flex vs grid adjustments, min-width vs explicit widths, shrink-0 placement)
- Whether to introduce a `@media (min-width: ...)` breakpoint
- Whether DPR handling refactors into a helper function

### Out of Scope
- New features, new components
- Behavioral changes to WebSocket, store, replay controller
- Anything outside `dashboard/`
- Operator smoke test sign-off (covered by Phase 11 Task 3)

</decisions>
