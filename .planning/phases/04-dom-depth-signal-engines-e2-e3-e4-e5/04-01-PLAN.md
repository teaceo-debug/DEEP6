    - TRAP-05 (CVD trap): cvd_history is a rolling deque. CVD has been trending (slope > 0 for N bars) then suddenly reverses direction and bar closes against the trend. Uses numpy polyfit(range(n), cvd_values[-cvd_trap_lookback:], 1) — slope[0] sign flip. Direction = sign of reversal.
    - All variants: strength = 0.0-1.0 based on signal quality metrics. Return empty list if bar.total_vol == 0 or data insufficient.

  <behavior>
    - VOLP-01 (volume sequencing): 3+ consecutive bars where each bar's total_vol is >= prior by vol_seq_step_ratio (default 1.15). Indicates institutional accumulation/distribution. Direction = sign of dominant bar_delta in the sequence.
    - VOLP-02 (volume bubble): single price level within the bar whose (bid_vol + ask_vol) > bar_avg_level_vol * bubble_mult (default 4.0). bar_avg_level_vol = bar.total_vol / len(bar.levels). Fire one signal per bar at the bubble price.
    - VOLP-03 (volume surge): bar.total_vol > vol_ema * surge_mult (default 3.0). Direction = sign of bar.bar_delta if |bar_delta/total_vol| > 0.15, else 0.
    - VOLP-04 (POC momentum wave): POC has migrated in same direction for poc_wave_bars (default 3) consecutive bars. poc_history is list of POC prices. Direction = sign of (poc_history[-1] - poc_history[-poc_wave_bars]).
    - VOLP-05 (delta velocity spike): compute velocity = bar.bar_delta - prior_bar.bar_delta. If |velocity| > vol_ema * delta_velocity_mult (default 0.6), fire. Direction = sign of velocity.
    - VOLP-06 (big delta per level): find the single level with highest |net_delta| (ask_vol - bid_vol). If that level's |net_delta| > big_delta_level_threshold (default 80 contracts), fire. Direction = sign of net_delta at that level.
    - All: return [] if bar.total_vol == 0 or insufficient history.
  </behavior>
