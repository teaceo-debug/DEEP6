- D-01: Weight = 1/level_index (level 0 = weight 1.0, level 1 = 0.5, level 2 = 0.33, ...)

    - process(dom_snapshot) where dom_snapshot is the tuple from DOMState.snapshot():
      weighted_bid = sum(sizes[i] * weights[i] for i in range(N)) where N = min active levels
      weighted_ask = sum(ask_sizes[i] * weights[i] for i in range(N))
      imbalance_ratio = weighted_bid / weighted_ask if weighted_ask > 0 else 0.0
      direction = +1 if ratio > 1.2, -1 if ratio < 0.8, else 0
      probability = logistic(log(ratio)) approximated as: min(max((ratio - 1.0) * 0.5 + 0.5, 0.0), 1.0)
    - process(None) returns TrespassResult(imbalance_ratio=1.0, direction=0, probability=0.5, detail="DOM_UNAVAILABLE")
    - depth_gradient: (bid_sizes[0] - bid_sizes[trespass_depth-1]) / (trespass_depth) — measures thinning of book
    - TrespassResult fields: imbalance_ratio: float, direction: int, probability: float, depth_gradient: float, detail: str
    - process must execute in < 0.1ms — use pre-computed weight array, avoid string formatting in hot path (build detail string only if debug=True)

  <behavior>
    - CounterSpoofEngine maintains internal state:
      - `_snapshot_history: deque` of (timestamp, bid_sizes_list) pairs — maxlen = spoof_history_len (default 20)
      - `_w1_history: deque` of float — rolling W1 distances, maxlen = spoof_history_len
      - `_level_timestamps: dict[float_price, (size, timestamp)]` — tracks large orders per level
    - ingest_snapshot(bid_prices, bid_sizes, ask_prices, ask_sizes, timestamp): called from asyncio timer every 100ms
      - Extracts bid distribution: list of (price, size) tuples for all levels where size > 0
      - If _snapshot_history not empty: compute W1 distance between current and previous bid distributions
        W1 = scipy.stats.wasserstein_distance(prev_sizes_array, curr_sizes_array)
        Append W1 to _w1_history
      - Append current snapshot to _snapshot_history
      - Cancel detection: compare current bid_sizes against _level_timestamps for large orders:
        For each level where _level_timestamps has size > spoof_large_order (default 50):
          If current size < spoof_cancel_threshold (default 10) AND (timestamp - recorded_ts) < spoof_cancel_window_ms/1000:
            Create SpoofAlert for that level
        Update _level_timestamps: for each level with size > spoof_large_order, record (size, timestamp)
    - get_w1_anomaly() -> float | None: returns current W1 distance if > mean + 3*std of _w1_history, else None
      - Returns None if _w1_history has fewer than w1_min_samples (default 5) entries
    - get_spoof_alerts() -> list[SpoofAlert]: returns alerts generated since last call, then clears buffer
    - SpoofAlert fields: price: float, prior_size: float, current_size: float, elapsed_ms: float, detail: str
    - No blocking operations. ingest_snapshot is called from asyncio (or a timer task) — must complete < 0.5ms.
    - All state is per-instance — no globals.
  </behavior>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| DOM callback → CounterSpoofEngine | ingest_snapshot called from asyncio timer task, not from 1000/sec callback |
| External DOM data | bid/ask sizes are floats from array.array — must handle zeros and negative values gracefully |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | Denial | TrespassEngine div-by-zero on empty DOM | mitigate | Guard `if weighted_ask == 0: return neutral_result` |
| T-04-05 | Denial | CounterSpoofEngine scipy W1 on empty arrays | mitigate | Pad both arrays to length >= 1 before wasserstein_distance call; handle len=0 case |
| T-04-06 | Denial | W1 std=0 when all values identical | mitigate | Guard `if std < 1e-9: return None` in get_w1_anomaly() |
| T-04-07 | Tampering | _level_timestamps dict grows unbounded | mitigate | Prune levels not seen in last 5 snapshots; cap dict at LEVELS entries |
</threat_model>
