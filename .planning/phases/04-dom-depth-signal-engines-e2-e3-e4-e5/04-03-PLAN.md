    - check_trade(price, size, aggressor_side, dom_snapshot, timestamp) -> IcebergSignal | None:

        After each DOM snapshot update, compare current size vs prior size for each level:
          If prior_size > iceberg_min_size (default 30) AND current_size < iceberg_min_remaining (default 5):
            Record depletion timestamp for that price level
          If current_size >= prior_size * iceberg_refill_ratio (default 0.80) AND level has a depletion timestamp within iceberg_refill_window_ms (default 250ms):
            Increment _refill_counts[price]
            If _refill_counts[price] >= iceberg_min_refills (default 2): fire SYNTHETIC iceberg

    - update_dom(bid_prices, bid_sizes, ask_prices, ask_sizes, timestamp) -> list[IcebergSignal]:
      Call after each DOM snapshot to advance synthetic detection state.
      Returns list of new IcebergSignals from refill detection. Empty list if none.

      `mark_absorption_zone(price: float, radius_ticks: int = 4) -> None`
      `is_at_absorption_zone(price: float) -> bool`
      conviction_bonus = 3 if is_at_absorption_zone(price) else 0

    - reset() -> None — clears all internal depletion/refill state (call at session start)

    - MicroEngine combines three binary features (per D-11):
      Feature 1: trespass_direction — encoded as: 1=bull, -1=bear, 0=neutral
        From TrespassResult.direction (or 0 if TrespassResult is None)
      Feature 2: iceberg_present — bool: any iceberg in current iceberg_signals list
        Direction of iceberg if present, else 0
      Feature 3: imbalance_direction — int: +1/-1/0 from narrative imbalance signals
        (Dominant imbalance direction: most buy imbalances vs sell imbalances)
    - Naive Bayes without training (heuristic prior per D-02 note):
      P(bull) starts at 0.5 (uniform prior)
      For each feature:
        If feature == +1: multiply P(bull) by bull_likelihood (default 0.65)
        If feature == -1: multiply P(bull) by (1 - bull_likelihood) = 0.35
        If feature == 0: no update (uninformative)
      Normalize: final_prob = P_bull_product / (P_bull_product + P_bear_product)
      where P_bear_product starts at 0.5 and updates symmetrically
    - MicroResult fields: probability: float, direction: int, feature_count: int, detail: str
      direction = +1 if probability > 0.6, -1 if < 0.4, else 0
      feature_count = number of non-zero features used
    - process(trespass: TrespassResult | None, iceberg_signals: list[IcebergSignal], imbalance_direction: int) -> MicroResult
    - process(None, [], 0) returns MicroResult(probability=0.5, direction=0, feature_count=0, detail="DOM_UNAVAILABLE") per D-13
    - No global state. No sklearn dependency — pure Python math (no fit() needed for heuristic prior).
  </behavior>
