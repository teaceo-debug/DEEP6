Helper to create synthetic bars for testing (replicate from existing test files):
```python
from collections import defaultdict
from deep6.state.footprint import FootprintBar, FootprintLevel, price_to_tick

def make_bar(open_=100.0, high=100.75, low=99.75, close=100.5,
             poc_price=100.25, levels=None) -> FootprintBar:
    if levels is None:
        levels = {price_to_tick(100.0): FootprintLevel(bid_vol=50, ask_vol=100)}
    total = sum(l.bid_vol + l.ask_vol for l in levels.values())
    return FootprintBar(
        open=open_, high=high, low=low, close=close,
        total_vol=total, poc_price=poc_price,
        bar_range=high - low, levels=levels,
    )
```
