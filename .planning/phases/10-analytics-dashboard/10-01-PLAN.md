<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser→/ws | WebSocket upgrade from browser; token in query param or header |
| browser→/backtest | HTTP POST triggers CPU-bound job in executor thread |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01 | Spoofing | /ws endpoint | mitigate | WS_TOKEN bearer token check at connect; close(1008) on mismatch |
| T-10-02 | DoS | /backtest/run | mitigate | 409 gate prevents concurrent jobs; executor thread limit is 1 |
| T-10-03 | Information Disclosure | ws broadcast | accept | Single-operator system (D-24); all clients share same session |
| T-10-04 | DoS | ConnectionManager | mitigate | Broadcast catches SendError and disconnects dead sockets silently |
</threat_model>
