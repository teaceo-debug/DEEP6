**BacktestRunner.cs** — sealed class with `Run(BacktestConfig config, string[] ndjsonPaths)` returning `BacktestResult`:
1. For each ndjsonPath, call `CaptureReplayLoader.LoadScoredBars(path)` to get `IEnumerable<ScoredBarRecord>`
2. For each ScoredBarRecord:
   a. Call `ConfluenceScorer.Score(rec.Signals, rec.BarsSinceOpen, rec.BarDelta, rec.BarClose, rec.ZoneScore, rec.ZoneDistTicks, config.TickSize)`
   b. If currently in a trade, check exits IN THIS ORDER:
      - **Stop loss**: if direction=+1 and barClose <= entryPrice - (config.StopLossTicks * config.TickSize), or direction=-1 and barClose >= entryPrice + (config.StopLossTicks * config.TickSize) → exit "STOP_LOSS"
      - **Target**: if direction=+1 and barClose >= entryPrice + (config.TargetTicks * config.TickSize), or direction=-1 and barClose <= entryPrice - (config.TargetTicks * config.TickSize) → exit "TARGET"
      - **Opposing signal**: if scored.Direction != 0 and scored.Direction != tradeDirection and scored.TotalScore >= config.ExitOnOpposingScore → exit "OPPOSING_SIGNAL"
      - **Max bars**: if (rec.BarIdx - entryBarIdx) >= config.MaxBarsInTrade → exit "MAX_BARS"
   c. When exiting: exitPrice = barClose + (direction * config.SlippageTicks * config.TickSize * -1) [slippage against us], PnlTicks = (exitPrice - entryPrice) / config.TickSize * direction, PnlDollars = PnlTicks * config.TickValue * config.ContractsPerTrade. Create Trade and add to result.
   d. If NOT in a trade, call `ScorerEntryGate.Evaluate(scored, config.ScoreEntryThreshold, config.MinTierForEntry)`. If `Passed`:
      - entryPrice = scored.EntryPrice + (direction * config.SlippageTicks * config.TickSize) [slippage against us on entry: long buys higher, short sells lower]
      - Direction = scored.Direction
      - Record entryBarIdx, SignalId from first ABS/EXH in scored.CategoriesFiring or "MIXED", Tier, Score, Narrative
3. After all bars in a session, force-exit any open trade with ExitReason="SESSION_END" at last bar close
4. Return BacktestResult with all trades and config.InitialCapital
