    Algorithm (mirror deep6/engines/volume_profile.py aggregation):

    1. `OnBarClose`: if `barTimeEt.Date != CurrentSession.SessionDate` → call `OnSessionBoundary(barTimeEt.Date)` first. Merge `bar.Levels` into `CurrentSession.VolumeAtPrice`. Update H/L/Close/Open accumulators.
    2. `OnSessionBoundary`: compute POC = argmax(VolumeAtPrice.Value); compute VAH/VAL via the same 70% algorithm as `FootprintBar.ComputeValueArea` but over the session-aggregated dict. Append to `CompletedSessions`, cap list at `NakedPocMaxAgeSessions + 5`. For every prior completed session's POC, if `CurrentSession.Low <= POC <= CurrentSession.High` during aggregation → mark `Retested = true`. Start a new `CurrentSession` for `newSessionDate`.
    3. `BuildSnapshot`:
       - Prior day (CompletedSessions[^1]) → emit PriorDayPoc/PriorDayVah/PriorDayVal, Pdh=High, Pdl=Low, Pdm=Mid.
       - Prior week = aggregate VolumeAtPrice across last 5 completed sessions, emit PriorWeekPoc.
       - Naked POCs = every POC from completed sessions where `Retested == false` AND `AgeSessions <= NakedPocMaxAgeSessions`.
       - Composite VA = merge VolumeAtPrice across last `CompositeSessions` (default 5) completed sessions → apply 70% VA → emit CompositeVah/CompositeVal. Null if fewer sessions accumulated.
    4. `Reset` clears everything (for DEEP6Footprint `State.DataLoaded`).
